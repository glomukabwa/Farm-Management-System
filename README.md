# Farm-Management-System

## About
This is a **web application** that will be used to manage farm activities which include animal feeding, product sales and veterinary visits. It will help the users have a centralized platform to keep track of everything that is going on in their farm.

## Functionalities and Pages
- An overview page with a table to keep track of the feeding schedule, a bar graph to keep highlight of the weekly sales and an alert section to alert the user of personal reminders and dire things such as low stock of feeds
- A centralized page to update farm records eg updating daily tasks done like feeding
- A product page for each product with a report of the feeding, health and sales
- A calendar to schedule events and reminders
- A profile page
- A settings page
- Log out option
- On the admin side, a report to show employee performance and a centralized page to view all farm records

## Technologies Used
- FrontEnd: Html, CSS, JavaScript<br>
- Backend: PHP<br>
- Database: MySQL
